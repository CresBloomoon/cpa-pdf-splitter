# CPA PDF Chapter Splitter

公認会計士試験（CPA）の教材PDFを  
**章ごとに自動分割できる Web ツール**です。

---

## なぜ作ったか

CPA教材のPDFは、

- 章ごとに分けて使いたい  
- iPad / GoodNotes で「今の章だけ」見たい  
- でも **手作業でのPDF分割が地味に時間かかる**

という問題がありました。

特に、

- 章扉が画像だったり  
- 目次が混ざっていたり  
- 1ページずれて分割される事故  

これを**毎回手で確認・修正するのが面倒**だったので、

👉 **アップロード → プレビューで確認 → 問題なければ一発分割**

できるように自動化しました。

---

## 主な機能

- 📄 PDFをドラッグ＆ドロップでアップロード
- 🔍 章タイトルを自動検出（目次ページは除外）
- 👀 各章をページ単位でプレビュー可能
  - サムネイル表示
  - クリックで拡大確認
- ✂️ 章ごとにPDF分割
- 📦 ZIP形式でダウンロード
- 🐳 Docker完結（ローカル環境を汚さない）

---

## 使いどころ

- CPA教材（租税法・企業法・管理会計など）
- 章単位での復習・持ち運び
- iPad学習用PDFの整理

---

## 技術スタック

- Backend: **FastAPI (Python)**
- PDF処理: **PyMuPDF**
- Frontend: **HTML / CSS / Vanilla JS**
- Infrastructure: **Docker / Docker Compose**
